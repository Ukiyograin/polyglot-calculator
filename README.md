# polyglot-calculator
这是一个用16种编程语言实现的整活项目，每个语言负责不同的计算功能，通过文件系统协同工作，形成一个完整的计算器系统

## 注意
- 此项目只供娱乐、学习、交流，请勿用于生产环境
- vibe coding by [deepseek](https://www.deepseek.com/)
  
## 需要环境
- Python
- JavaScript - Node.js
- Java - JDK 8+
- C# - Mono或.NET Core
- Ruby - Ruby 2.7+
- Rust - Rustc
- Go - Go 1.16+
- Kotlin - Kotlin编译器
- PHP - PHP 7+
- Swift - Swift编译器
- Lua - Lua 5.3+
- Perl - Perl 5
- R - R语言环境
- TypeScript - Node.js + TypeScript
- MATLAB - MATLAB或Octave
- Bash - Bash shell
- SQLite - SQLite3（通常系统自带）
